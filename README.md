# Oceanix CSS

## Description

This is a *really modern* .css package that has a really cool combination of **color tones**! If you want even more styled, you will have to add more .css files, for example, using the font-size part, that Oceanix CSS doesnt put so you can make it how you want.

## For what do I use Oceanix CSS?

* Modern Webpages
* Chats with *AI*/Global Chats with friends
* *E-commerce* Webpages
